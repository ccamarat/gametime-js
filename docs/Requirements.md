## HMI
### Inputs
* ACL (All Clear)
* Power
* Pause / Play (toggle?)
* Select
* Verify
* Plus / White
* Minus / Counter
* Left Plunger
* Right Plunger

### Outputs
* LED Display
* Left LED (plunger)
* Right LED (plunger)

## Behavior
* When active:
  - All buttons disabled
  - Left plunger starts "Rightside" turn
  - Right plunger starts "leftside" turn
* When paused:
  - Plungers disabled
  - "White" button changes which side is "first"
  - "Select" button enables toggling settings
